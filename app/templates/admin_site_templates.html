站台抽取模板

← 返回控制台

這裡管理「單一網站專屬」的主文抽取模板——某網站文章內文在哪個 CSS 容器。 爬蟲對命中的網站直接用模板選擇器抓正文，比通用啟發式準。
內建模板為地板（安全基線，程式內，Firestore 空/掛時仍可運作）；此處 Firestore 可增/改/刪（以模板名為 key 覆寫或新增）。儲存後爬蟲最多 60 秒生效、不需重新部署。
加新站台：在 JSON 裡新增一個 "模板名": {"indicators": [...], "selectors": [...]}。 indicators＝命中此模板的 URL 片段（含網域更具體，如 cna.com.tw）； selectors＝由具體到通用、依序試的主文容器 CSS。

模板清單（目前 {{ n_templates }} 個，JSON）
格式：{ "模板名": { "indicators": ["網域或URL片段"], "selectors": ["CSS選擇器", ...] }, ... } 。請保持合法 JSON。
⚠️ 儲存會以此 JSON 整份覆寫 Firestore 模板。內建地板不受影響。
範例：
"cna": { "indicators": ["cna.com.tw"], "selectors": [".paragraph", "article .centralContent"] }
