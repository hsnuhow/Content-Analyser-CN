① 內容來源
把要分析的內容弄進專案，三種方式擇一： 🔍 關鍵字推薦（系統找代表性內容）／ 📋 貼上網址（自己有連結）／ 📥 上傳資料（已蒐集的檔案或文字）。產物都會成為下方的資料集。
▸ 🔍 關鍵字推薦（選用，點此展開）

輸入主題關鍵字，系統用 Google 搜尋情報找出該主題台灣現正排名的代表性內容 （媒體 / 論壇 / 影音 / 電商），結果存成推薦清單，勾選後建立草稿資料集。列表頁/首頁預設不勾。

❓ 什麼時候用（點開）
不知道要分析哪些內容時用它。你只給一個主題詞（消費者會搜的詞，如「循環扇」）， 系統幫你把「現在誰排在前面、大家在看什麼」的代表頁面找出來，你勾一勾就能去爬，省得自己 google 貼網址。
產生後會存成下方「推薦清單」，可隨時回來勾選 → 建立草稿資料集或併入現有草稿。

{# 品牌聲量探勘（搜尋情報：誰在這主題有 earned 聲量；找「有賣卻沒聲量」缺口）#} {% if role in ('owner', 'editor') %}
▸ 📊 品牌聲量探勘（誰在這主題有聲量）

主題 + 品牌清單 → 看每個品牌在台灣這主題有沒有「第三方」聲量（評測/開箱/討論）， 找出「有賣卻沒人談」的缺口。每行一個品牌（建議 ≤8 個，較快）。

❓ 怎麼用 / 怎麼看（點開）
這工具回答：「這個主題上，哪些品牌有人在談、哪些沒人理？」
怎麼用：① 填主題（消費者會搜的詞，如「循環扇」）→ ② 一行一個品牌（自家 + 競品）→ ③ 按「分析品牌聲量」，等幾十秒。
三個燈號怎麼看：
  • 有聲量　很多別人（媒體/論壇/YouTuber）在談 → 已有口碑
  • 僅自有　只有品牌自己的官網/賣場、別人沒談 → 有賣但沒人幫它說話
  • 缺席　幾乎找不到 → 在這主題上等於隱形
（「第三方」數字＝有幾個外部來源在談它，越大聲量越強。）
看到結果可以做什麼：
  • 自家是僅自有/缺席 → 該做內容/找 KOL/投放，把聲音做出來。
  • 競品有聲量、你缺席 → 點開它的來源，看它在哪被談，照著補。
  • 大家都缺席 → 藍海，先做先贏、搶當代表內容。
  • 找客戶：僅自有的品牌＝「在賣卻沒聲量」，正是需要內容行銷的對象。
  • 驗證成效：活動前後各跑一次，看有沒有從 🟡/🔴 變 🟢。
小提醒：這是快速研判、不是精準普查，每次略有不同；要更確定就多跑一兩次看趨勢。

❓ 溫度、top_p 是什麼（點開）
這兩個都在控制 AI「多保守 vs 多發散」：
  • 溫度（創意度）：0＝每次都穩、照數據講、可重複；越高＝越敢發揮但也越可能跑題或每次不一樣。 做分析報告建議 0.2–0.4（要穩、要能對照）。寫行銷文案才調高。
  • top_p（進階區，留空＝用預設）：另一個控發散的旋鈕，越低 AI 用詞越聚焦、越高越多樣。 一般留空就好，別跟溫度同時調高，否則報告每次差很多。
註：這裡是 top_p，不是 top_k；兩者概念類似（都在限制 AI 選字的範圍），本系統用的是 top_p。
