我們的方法論
  • 市場基準線：會紅代表需求真實存在 → 拆解共同模式，知道「做什麼方向對」。
  • 差異化切點：找「受眾在意、卻沒被說好」的落差 → 這就是你的競爭位置。
四步開始
  1. 建專案，填你的 LLM API Key（Gemini／Claude 自備）
  2. 爬一批受歡迎內容的網址，成為資料集
  3. 一鍵送分析
  4. 看報告 + 一鍵產生延伸行動指引（AEO／品類經理／投放師）
💡 挑「市場已驗證表現好」的內容（Google 前三頁、高互動影片／貼文）；同一主題多來源更準； 報告裡「用戶搜尋情境」與「內容缺口」這兩段最值得先看。
